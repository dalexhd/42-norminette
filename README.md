# 42 Norminette Checker

This extension will inject norm errors inside your code.

![demo image](demo.gif)

## Configuration

```ts
{
  "42norminette.command": string, // Default: 'norminette'
  "42norminette.maxErrors": number // Default: 1,
}
```
